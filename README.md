# 光影交错 (Light & Shadow Intersection)

一个基于**光学加法混色**原理的益智解谜游戏。在棋盘上放置红、绿、蓝三色光源，光束交汇产生新颜色，照亮所有目标靶心即可过关。

## 🎮 在线试玩

👉 **[点击这里立即开始游戏](https://tommao46.github.io/light-and-shadow/)**

## 🧠 游戏机制

| 机制 | 说明 |
|------|------|
| 🔴🟢🔵 **光源** | 放置红/绿/蓝光源，向上、下、左、右四个方向射出光束 |
| 🎨 **混色** | 红+绿=黄 / 红+蓝=品红 / 绿+蓝=青 / 三色=白 |
| 🧱 **墙壁** | 阻挡光线，迫使你寻找新路径 |
| 🔄 **反射镜** | 光束碰到镜面**双向反射**90°，绕过障碍 |
| 🔍 **滤光片** | 只允许**指定颜色**通过，其余被阻挡 |
| 🎯 **目标靶心** | 每个靶心需要被特定颜色的光照亮 |

## 🕹️ 操作

| 操作 | 方式 |
|------|------|
| 选择光源 | 点击右侧面板 或 按键盘 `R` `G` `B` |
| 放置光源 | 点击棋盘空格 |
| 移除光源 | 右键点击 或 再次点击已放置的光源 |
| 重置关卡 | 点击「重置本关」或按 `Backspace` |
| 获取提示 | 按 `H` 键 或等待提示按钮出现 |
| 切换关卡 | `Ctrl + ← →` |
| 悬停预览 | 选中光源后鼠标悬停在格子上预览光束 |

## 📊 20 关难度递进

| 区间 | 关卡 | 重点 |
|------|------|------|
| 1-6 | 初见光芒 ~ 纯白之光 | 光线基础 + 颜色混合 |
| 7-10 | 墙壁迷宫 ~ 共享绿光 | 墙壁封锁 + 路径规划 |
| 11-14 | 对角线 ~ 八阵图 | 大网格 + 多目标协同 |
| 15-17 | 暗角红光 ~ Z形折返 | 反射镜双向偏转 |
| 18-19 | 滤光初试 ~ 三色分滤 | 滤光片颜色筛选 |
| 20 | 终极织光 | 反射镜 + 滤光片 + 墙壁三合一 |

## ⏱️ 计时与记录

- 每关独立计时，通关后显示用时
- **最快记录**永久保存（localStorage）
- 打破纪录时弹出 🏆 提示

## 🛠 技术

- 纯前端：HTML + CSS + JavaScript
- Canvas `globalCompositeOperation: 'lighter'` 实现真实加法混色
- 光线传播算法支持反射变向和滤光阻挡
- localStorage 保存进度和最佳时间
